# .claude/skills/

Reusable task templates. Skills are longer-form prompts for multi-step workflows that appear too often to re-type but are too complex for a simple slash command.

## Adding a skill

1. Create `<skill-name>.md` here.
2. Write a structured prompt with clear inputs, steps, and expected outputs.
3. Document it in this README.

## Current skills

None defined yet. Candidates to add as patterns solidify:

- `scaffold-service` — spin up a new service following team conventions
- `migrate-framework` — structured approach for large framework upgrades
- `add-mcp` — checklist for wiring a new MCP server
