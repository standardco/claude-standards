# .claude/commands/

Shared slash commands available in all projects that import this repo.

Slash commands live as markdown files. The filename becomes the command name: `review-pr.md` → `/review-pr`.

## Adding a command

1. Create `<command-name>.md` in this directory.
2. Write the prompt the command expands to.
3. Use `$ARGUMENTS` anywhere you want the user's inline arguments inserted.
4. Document it in this README.
